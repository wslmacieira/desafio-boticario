# Desafio Boticario

### Objetivo:

- Desenvolver um sistema 
- Cadastro de revendedor
- Cadastro de compra
- Listagem de compras cadastradas e total de cash e status da compra
- Login com senha e email


### Techs


* node.js - no backend
* ReactJS - n frontend
* Typescript


### Installation

Install the dependencies and devDependencies and start the server.

```sh
$ cd backend
$ npm install -d 
$ yarn
```

For frontend 

```sh
$ cd frontend
$ npm install 
$ yarn install
```
